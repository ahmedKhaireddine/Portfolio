## 🏗 Installation & Setup

```bash
# 1. Clone the repository
git clone https://github.com/ahmedKhaireddine/portfolio.git

# 2. Navigate to the project directory
cd portfolio

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```
